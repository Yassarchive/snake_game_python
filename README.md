# Snake Game (Python + Pygame)

Een klassieke Snake-game gemaakt met Python en de Pygame-library.

## 🎮 Overzicht

Dit project simuleert de klassieke Snake-game waarin je een slang bestuurt die groeit door voedsel te eten. Het doel is om zo lang mogelijk te blijven zonder tegen jezelf of de rand te botsen.

## 🚀 Features

- Besturing met pijltjestoetsen
- Score wordt weergegeven in de titelbalk
- Dynamische groei van de slang
- Eenvoudige game loop met Pygame

## 🛠️ Technologieën

- Python 3.12.3
- Pygame
